Sobre a loja

Nossa loja é aberta a todos os grupos da comunidade gamer, mas a nossa atenção é voltada para o público feminino dentro dessa comunidade. Promovemos eventos para este público como por exemplo, campeonatos femininos e unissex! Possuimos artigos gamer especializado para as meninas, como mouses gamer menores, headsets de alta qualidade mas com aquele charme que todas amamos mas sem deixar de lado o conforto. Apoiamos causas femininas e todo semestre a ganhadora do nosso campeonato GIRL GPOWER é premiada com perifericos durante um ano e se torna nossa garota propaganda durante esse periodo.

Oferecemos produtos de ponta e os mais atuais do mercado, com preços acessíveis a todos os bolsos desde jogos para consoles a periféricos para voce montar aquele seu set up dos sonhos.
